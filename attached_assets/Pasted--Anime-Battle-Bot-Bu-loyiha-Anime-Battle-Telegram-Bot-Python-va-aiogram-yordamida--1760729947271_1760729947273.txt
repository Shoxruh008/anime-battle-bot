## ⚔️ Jang tizimi

### 🎯 Jang bosqichlari:

1. Foydalanuvchi jang turini tanlaydi: `CPU` yoki `Real`.
2. Har ikkala tomon o‘z **komandasini** tanlaydi (3 yoki 5ta personaj).
3. Kim birinchi yurishini **random** aniqlanadi.
4. O‘yin davomida:

   * Foydalanuvchi **Attack**, **Defend**, yoki **Special** tanlaydi.
   * CPU esa **aqlli tizim asosida** javob beradi (faqat random emas).
   * Attack va Defense kuchlari `characters.json`dagi statistikaga asoslanadi.
5. Yutgan foydalanuvchi:

   * `Anicoin` oladi (raqib kuchiga qarab miqdor belgilanadi)
   * Premium jangda yutgan bo‘lsa — `Battlecoin` ham oladi.
   * Yutqazgan — `Anicoin`ini yo‘qotadi.

## 🧩 Kod namunasi (aiogram)

```python
from aiogram import Bot, Dispatcher, executor, types
import json, random

bot = Bot(token="7995099850:AAFaan-VTbWJtuDKVLQoL4Yk4nLVCz7jxgU")
dp = Dispatcher(bot)

@dp.message_handler(commands=['start'])
async def start_game(msg: types.Message):
    await msg.answer(
        "🌀 Anime Battle Bot'ga xush kelibsiz!\n\n"
        "Tanlang:\n"
        "🃏 Karta olish\n📦 Mening kartalarim\n⚙️ Menyu\n👤 Profil"
    )

@dp.message_handler(lambda m: m.text == "🃏 Karta olish")
async def get_card(msg: types.Message):
    with open("data/characters.json", "r") as f:
        data = json.load(f)
    card = random.choice(data)
    await msg.answer(f"Sizga {card['name']} ({card['anime']}) chiqdi!\nAttack: {card['attack']}, Defense: {card['defense']}")

if __name__ == "__main__":
    executor.start_polling(dp, skip_updates=True)
```
